Csharp Intro

C#是一类面向对象的编程语言.扳手图标意味着该类下的一个属性或者变量，而盒子图标意味着该类下的一个方法(函数).

  • 函数在调用时需要在后面加一个"()"来填入一些参数,当然,一些函数可能不需要参数,但是括号还是必要的.例如:
    System.Console.ReadLine();
  • 属性在修改时需要使用赋值语句"=",等号右边就是要新赋予的值.例如:
    System.Console.Title = "My Console";

注意
编写 C#代码时，每一个语句后面都要加一个";"来表示语句的结束！

Variables

变量(Variables)可以用来存储一个数据,以便以后随时调用.

  • 变量的声明
    Type name ;
  • 变量的数据类型
    变量可以存储数据,然而一个变量只能存储一种类型的数据.不同的数据有着不同的处理方式.下面是一些常见的数据类型.
    • int 整数类型
    • string 字符串类型(即文本) char 字符类型(即单个字符)
    • double float 浮点类型(即小数)
    • bool 布尔类型(只有 True 或者 False)

注意
int age = 10;此时变量 age 是一个整数类型变量.
string _age = "10";此时变量_age 是一个字符串类型变量.

  • 调用变量的数据
using System;
namespace maincode{
	class Program{
		static void Main(String[] args){

If&Else switch-case

if else 语句可以根据条件来判断是否执行下面的语句.

  • if 语句创建
    if(conditon)
    {
    I want to run this code if the conditon is true;
    }

if 的一些具体知识
== 等于 && 且 || 或
!= 不等于
>/< 大于小于
if 就是根据这些判断来给条件赋予"True"或者"False".若 True,就执行 if 的代码

当然,if 的格式没有那么严格,这不是 python.上面这么写只是为了美观.

  • 用 if 语句来实现情景判断
class Program
{
	static void Main(string[] args)
	{
	Console.WriteLine("Welcome! Tickets are 5$. Please insert cash.");
	int cash = Convert.ToInt32(Console.ReadLine());
	if(cash<5){
		Console.WriteLine("That's not enough money.");
	}
	else if (cash == 5){
		Console.WriteLine("Here is your ticket.");
	}
	else
	{
		int change = cash -5;
		Console.WriteLine("Here is your ticket and "+ change +" dollars in change.");
	}
	Console.ReadKey();
	}
}
  • switch-case 结构分支(选学)
    有的时候我们仅根据一个变量的值来判断分支,这个变量可能有几个不同的值,在这种情况下,我们可以用 switch-case 结构进行编写.
    下面是它的一个实际写法:
using System;

namespace codemain
{
    class Program
    {
        static void Main(String[] args)
        {
            Console.Title = "C# Test";
            Console.WriteLine("Type a number!");
            int num = Convert.ToInt32(Console.ReadLine());
            switch(num)
            {
                case 1:
                    Console.WriteLine("Yeah,it is 1");
                    break;
                case 2:
                    Console.WriteLine("Yeah,it is 2");
                    break;
                case 3:
                    Console.WriteLine("Yeah,it is 3");
                    break;
                default:
                    Console.WriteLine("I don't know.");
                    break;
            }
            Console.ReadKey();
        }
    }
}

Function&Method

函数或者叫做方法,可以让同一段代码重复调用.对于代码的整洁和整理有重要作用.

  • 函数的声明
    返回值类型 名字(传入的参数)
    {
    代码;
    }
    当然,有的函数没有参数需求,不写参数也可以.
  • 函数的调用
    回到 Main 函数,在这里写上名字();即可调用函数.

Main
实际上,Main 也是一种函数,它是程序的主入口,每次运行程序时,程序会优先寻找 Main 方法并调用.

using System;

namespace maincode
{
    class Program
    {
        static void Main(String[] args)
        {
            Console.Title = "C# Test";
            for (int i = 0; i < 10; i++)
            {
                PrintNumber();//调用函数
            }
            Console.ReadKey();
        }
        static void PrintNumber()
        {
            Random random = new Random();
            int num = random.Next(0, 10000);
            Console.WriteLine("生成一个随机数字中...");
            Console.WriteLine(num);
        }
    }
}
  • 现在,让我们在函数中添加参数,并让他把处理的值返回(return)出来.
using System;

namespace maincode
{
    class Program
    {
        static void Main(String[] args)
        {
            Console.Title = "C# Test";
            int result = PrintNumber(10);
            Console.WriteLine(result);
            Console.ReadKey();
        }
        static int PrintNumber(int number)
        {
            Random random = new Random();
            int num = random.Next(0, 10000);
            Console.WriteLine("生成一个随机数字中...");
            number = num + number;
            return number;
        }
    }
}

这里的 PrintNumber 函数,需要传入一个 int 类的数据,记作 number(只是占位,并没有实际意义),并最后把这个值返回出来,类型为 int,这也对应了它是一个 int 函数.

函数的参数
函数可以有多个参数填入,填入的参数均可以在函数内调用,它们都类似于占位的存在,只是一个形式,并没有实际的意义.
函数的类型取决于它的返回值,如果返回一个 string,那它就是一个 string 函数,返回一个string[]它就是一个字符串数组函数.

  • 计算一个字符串的长度(用字符串的 Split 方法实现)
using System;

namespace maincode
{
    class Program
    {
        static void Main(String[] args)
        {
            Console.Title = "C# Test";
            Console.WriteLine("输入一个句子,计算它的单词数.");
            int result = WordCount(Console.ReadLine());
            Console.Write(result);
            Console.ReadKey();
        }
        static int WordCount(string Text)
        {
            int Words = Text.Split(' ').Length;
            Console.WriteLine("这句话的单词数:");
            return Words;
        }
    }
}

Class&Objects

C#是一个面向对象的语言,万物皆可为对象.

  • 类的概念
    类可以视作是万物抽象化后结果,是具体对象的"蓝图".
    在类中,我们用代码去描述这些对象所共有的特征,然后再"实例化",创建一个具体的对象并调用它的方法实现功能.
  • 类的创建
    类的等级和 Program 一样,注意创建类的位置
using System;
namespace maincode
{
    class Program
    {
        static void Main(String[] args)
        {
            Console.Title = "C# Test";
            Cat Tom = new Cat();
            Cat Sam = new Cat();
            Tom.Name = "Tom";
            Sam.Name = "Sam";
            Tom.Meow();
            Sam.Meow();
            Console.ReadKey();
        }
    }
    class Cat
    {
        public string Name = "";
        public int Age;
        public void Meow()
        {
            Console.WriteLine(Name + ":Meow!");
        }
    }
}

要实例化一个类,用 new 方法即可.创建后,用名字.的形式就可以调用这个对象的变量和方法

  • 构造函数
    每次创建一个实例对象都要写好几行代码填写参数是很烦的,我们可以利用构造函数简化这个过程.
using System;

namespace maincode
{
    class Program
    {
        static void Main(String[] args)
        {
            Console.Title = "C# Test";
            Cat Tom = new Cat("Tom",2);
            Console.WriteLine(Tom.name + " is " + Tom.age +".His favorite food is " + Tom.favoriteFood);
            Console.ReadKey();
        }
    }
    class Cat
    {
        public string name = "";
        public int age;
        public string favoriteFood = "";
        public Cat(string _name,int _age)//创建构造函数
        {
            name = _name;
            age = _age;
            favoriteFood = "Fish";
        }
        public void Meow()
        {
            Console.WriteLine(name + ":Meow!");
        }
    }
}

修饰符

Static 意为静态的，被该关键字修饰的变量或者函数，会被归属于类本身而非某一个 实例对象上.

Private 只能在类内访问，不能在该类以外的地方进行访问或者修改; Public 可以在类内类外随意访问，修改;

如果不加访问修饰符，则该类默认为 private

< >为泛型,如果在定义函数或者类的时候,其中有多个数据类型相同的变量,可以直接用< >"泛泛而谈"全部概括

例如:
PlusInt<int>(1,5);

泛型可以省略很多麻烦的备注!
